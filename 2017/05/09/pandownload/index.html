百度网盘下载助手脚本 — 有效，2017-05-09 更新

发表于 2017-05-09
字数统计 471 | 阅读时长 2

各大网盘关闭后，百度网盘似乎成为国内网盘的唯一选择，而百度网盘却时常限速，下面方法抗压解决百度网盘限速的问题，演示的下载文件是大于1G的一个 War3.zip (单击打开)，使用的宽带是电信20M校园网，百度限速后的下载速度只有200+KB/s，而理论上的下载速度是可以达到2M/s以上。

百度网盘下载助手脚本 — 有效，2017-05-05 更新

最早是网友“有一份田”制作的脚本百度下载助手，可以显示直链，绕过大文件云盘下载，还可以调用迅雷或者IDM加速下载。

需要注意的是，在调用迅雷或IDM下载时，需要点击“显示直链”，然后再点击下载，支持在百度网盘 个人网盘管理页面 和 分享页面的下载。Firefox下可以安装拓展“xthunder”来调用下载软件，加速效果几近满速。


Firefox 浏览器需要安装拓展 Greasemonkey
Chrome 和360等浏览器需要安装拓展Tampermonkey。

脚本下载地址：https://greasyfork.org/zh-CN/scripts/23635

迅雷9官网下载 : http://down.sandai.net/thunder9/Thunder9.1.31.736.exe
IDM绿色版下载：https://pan.baidu.com/share/init?shareid=3457730696&uk=1767948507#3edk 访问码3edk

注意

1、IDM绿色版需要点击运行文件 “!绿化卸载.bat” 才会自动关联下载。
2、如果下载速度被限制在100KB左右，说明你的帐号被百度网盘拉黑了，需要退出登录才能恢复正常下载。
3、脚本更新的方法是打开附加组件页面，右键选择本脚本，然后点击更新，或者打开 脚本地址 重新下载覆盖安装。
4、文章更新时的最新版本为：0.9.23 (2017-05-05 更新)
